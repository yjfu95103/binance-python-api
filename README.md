# binance-python-api
Using Binance official api to display your currency distribution.

## Environment setup
    * create & launch env
    ```
    $ pip3 install pipenv  #install lib
    $ pipenv --python 3.8
    $ pipenv shell         #launch env
    ```
    * quit
    ```
    $ exit                 
    ```
